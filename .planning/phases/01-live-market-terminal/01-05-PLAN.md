| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-05-01 | Tampering | add-ticker input | medium | mitigate | Client uppercases and defers authoritative validation to the server (Plan 04); malformed input yields an inline 400 message, never a client-trusted mutation |
| T-05-02 | Denial of Service | Lightweight Charts instances per row | medium | mitigate | Create one chart/series per row in a ref and dispose on unmount; feed the already-bounded usePriceHistory series (≤600 pts) so per-row memory is capped |
| T-05-03 | Injection (XSS) | rendering ticker/price text | low | mitigate | Values rendered as React text nodes (auto-escaped); no dangerouslySetInnerHTML |
| T-05-SC | Tampering | package installs | low | accept | No new packages — lightweight-charts was installed and verified in Plan 03's legitimacy checkpoint |
